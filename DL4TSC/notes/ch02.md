## PyTorch 中的基本操作

我们可以使用各种技术在 PyTorch 中创建张量。让我们从列表创建张量开始：

```python
import torch

t1 = torch.tensor([1, 2, 3])
print(t1)
t2 = torch.tensor([[1, 2], [3, 4]])
print(t2)
```

PyTorch 可以与 NumPy 无缝集成，从而可以轻松地从 NumPy 数组创建张量：

```python
import numpy as np

np_array = np.array([5, 6, 7])
t3 = torch.from_numpy(np_array)
print(t3)
```
